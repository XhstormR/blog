纯 UEFI 模式下转换 GPT 格式#

1. BIOS 设置中启动项关闭兼容模式
2. 插入 Win8 启动盘后，出现第一个"下一步"，按shift+f10调出命令对话框
3. 输入 diskpart
4. 输入 select  disk 0
5. 输入 clean
6. 输入 convert gpt

到这里就可以关闭命令对话框，按下一步进行正常的分区设置了，系统会自动分出三个区用于UEFI的启动。
