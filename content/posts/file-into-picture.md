```

Windows Registry Editor Version 5.00
[HKEY_CLASSES_ROOT\*\shell\用 7z 打开]
"Icon"="D:\\Program Files\\7-Zip\\7zFM.exe"

@="\"D:\\Program Files\\7-Zip\\7zFM.exe\" \"%1\""
```
