Updated on 2019-10-03

> https://github.com/tmux/tmux

```bash
wget https://github.com/tmux/tmux/archive/master.zip
unzip master.zip
cd tmux-master/

yum groups install 'Development Tools'
yum install -y ncurses-devel libevent-devel

sh autogen.sh
./configure && make install
```
